## 主要仓库

- [`HaloForgeAI/HaloForge`](https://github.com/HaloForgeAI/HaloForge)：桌面端 runtime、插件加载器和本地应用外壳。
- [`HaloForgeAI/haloforge-plugin-api`](https://github.com/HaloForgeAI/haloforge-plugin-api)：Rust API crate、TypeScript SDK、packager、模板和插件开发 skill。
- [`HaloForgeAI/HaloForge-Community-Cloud`](https://github.com/HaloForgeAI/HaloForge-Community-Cloud)：公开 Cloudflare Workers 与 Pages 部署面。
- [`HaloForgeAI/HaloForge-Enterprise-Server`](https://github.com/HaloForgeAI/HaloForge-Enterprise-Server)：企业版部署和私有控制面代码。
